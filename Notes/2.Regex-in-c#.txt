What is Regex in C#? (Simplified Explanation)
----------------------------------------------

Regex (short for Regular Expressions) is a pattern-matching system used to search, validate, and manipulate text in C#.

Think of it as a smart filter that helps you find specific patterns in strings. For example:

Check if an email is valid ‚úÖ
Find all phone numbers in a document üìû
Replace certain words in a sentence ‚úèÔ∏è
In C#, regex is handled using the System.Text.RegularExpressions namespace.

1Ô∏è‚É£ Basic Example of Regex in C#
----------------------------

Let's check if a string contains a valid email:

using System;
using System.Text.RegularExpressions;

class Program
{
    static void Main()
    {
        string email = "test@example.com";
        string pattern = @"^[^@\s]+@[^@\s]+\.[^@\s]+$";  // Email pattern

        bool isValid = Regex.IsMatch(email, pattern);
        Console.WriteLine(isValid); // Output: True
    }
}

‚úÖ Regex.IsMatch(text, pattern) ‚Üí Returns true if the string matches the pattern.
‚úÖ The pattern checks if the string follows an email format.

2Ô∏è‚É£ Common Regex Methods in C#
------------------------------

Method					Description					Example
--------------------------------------------------------------------------------------------------------------------------------------------
Regex.IsMatch()			Checks if a string matches the pattern		Regex.IsMatch("hello123", @"\d") ‚Üí True
Regex.Match()			Finds the first match				Regex.Match("abc123xyz", @"\d+") ‚Üí "123"
Regex.Matches()			Finds all matches				Regex.Matches("abc123xyz456", @"\d+") ‚Üí ["123", "456"]
Regex.Replace()			Replaces matched text				Regex.Replace("Hello 123", @"\d", "*") ‚Üí "Hello ***"

4Ô∏è‚É£ Regex Cheat Sheet
----------------------

Pattern					Meaning					Example
--------------------------------------------------------------------------------------------------------------
\d				Any digit (0-9)				\d+ ‚Üí "123" in "Order 123"
\w				Any letter or number			\w+ ‚Üí "Hello" in "Hello 123"
\s				Any whitespace (space, tab, newline)	\s+ ‚Üí " " in "Hello World"
.				Any character except new line		a.b ‚Üí Matches "acb", "a_b"
^				Start of string	^Hello ‚Üí Matches 	"Hello world" but not "Hi Hello"
$				End of string	world$ ‚Üí Matches 	"Hello world" but not "world hello"
[]				Match any character inside		[aeiou] ‚Üí Matches any vowel
{n,m}				Match n to m repetitions		\d{2,4} ‚Üí Matches "12", "123", "1234"
